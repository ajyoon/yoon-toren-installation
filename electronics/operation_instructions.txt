Operation Instructions
======================

These are instructions for how to operate the program.
For help setting up the program, see setup.txt.

To operate the drone program, simply run it with Python
from the command line.
------------------------------
cd path/to/drone/folder
python drone.py
------------------------------
Make sure that your input and output devices are connected 
before starting the program.

When the performance begins (Cue A), press Play.

On Cue K, press Pause.

At the end of Cue K, press Play again.

On Cue P, press Cue P.
